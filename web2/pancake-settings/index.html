Cấu hình Pancake (Web 2.0)

Quản lý token Pancake riêng cho Web 2.0 — độc lập với module web2-pancake/ cũ. Token lưu localStorage, không gửi lên server.

Tài khoản Pancake —

Quản lý nhiều tài khoản Pancake (mỗi FB login = 1 account). Lưu ở DB (Render) — đồng bộ mọi máy. Bấm Dùng để chọn account active trên máy này; Thêm tài khoản để lưu account mới.
Đang tải danh sách tài khoản…
Dán JWT token của tài khoản Pancake mới, hoặc bấm Lấy tự động nếu đang đăng nhập pancake.vn trong trình duyệt này (cần extension).

Admin theo Page —

Mỗi page có bao nhiêu tài khoản admin (gộp từ tất cả account ở mục Tài khoản phía trên) và bao nhiêu account dùng được ngay — tức số worker chạy song song khi Tăng comment. Account hết hạn token hoặc tắt sync không tính là dùng được.
Đang tổng hợp…
Bấm để lấy lại danh sách page admin của mọi account còn hạn trực tiếp từ token — sửa trường hợp account có quyền page nhưng chưa hiện (vd account mới / chỉ vừa gia hạn token). Cập nhật luôn cho cả “Tăng comment”.

Server realtime (WS) — chọn trang nhận comment —

Relay web2-realtime kết nối Pancake WS để đẩy comment livestream về 2 trang Live (desktop + mobile) không cần poll. Mỗi trang join riêng (pages:{id}) nên trang nào hết gói cước chỉ trang đó rớt — các trang còn lại vẫn nhận comment. Tick trang muốn nghe (mặc định bật hết), bỏ tick trang không cần.

JWT Token (Pancake login) Chưa có

Tự động (khuyên dùng): bấm Lấy tự động — extension sẽ đọc cookie jwt từ pancake.vn (cần đang đăng nhập pancake.vn trong trình duyệt này). Không cần mở tab, không cần copy tay.
Thủ công: mở pancake.vn → F12 → Console → dán copy(document.cookie.match(/(?:^|; )jwt=([^;]+)/)[1]) → Enter (token đã vào clipboard) → quay lại đây dán vào ô bên dưới.

Pages & access tokens —

Mỗi page cần có page_access_token riêng để gửi tin/comment qua Public API. Click Generate để Pancake cấp token cho page (cần JWT hợp lệ).
Cần JWT token trước khi load pages.

Vùng nguy hiểm

Xoá toàn bộ token (JWT + tất cả page tokens). Tab1 + native-orders sẽ mất chat cho tới khi cấu hình lại.

Token Pancake sắp hết hạn

Extension đọc cookie từ pancake.vn — cần đang đăng nhập pancake.vn trong trình duyệt này.
Hoặc lấy thủ công
  1. Mở pancake.vn và đăng nhập.
  2. F12 → tab Console.
  3. Dán đoạn dưới, Enter (token vào clipboard):
copy(document.cookie.match(/(?:^|; )jwt=([^;]+)/)[1])

Tự động gia hạn tài khoản

Lưu mật khẩu (mã hoá AES ở server) để server tự đăng nhập lại gia hạn token khi sắp hết hạn — không cần thao tác tay.

⚠ Server chưa cấu hình khoá mã hoá (PANCAKE_CREDS_KEY) — chưa lưu được.
