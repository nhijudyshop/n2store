Cài Print Bridge (Windows — 1 click)

Print Bridge giúp in thẳng ra máy in mạng (IP) không cần hộp thoại. Bản PowerShell KHÔNG cần cài Node — tải file .bat bên dưới, bấm đúp 1 lần: nó tự cài, chạy nền và tự bật mỗi khi mở máy.

  1. Bấm Tải file cài đặt (.bat) → lưu về máy POS.
  2. Bấm đúp file cai-print-bridge.bat (nếu Windows hỏi SmartScreen → "More info" → "Run anyway").
  3. Xong! Bridge chạy nền + tự bật khi khởi động. Quay lại đây bấm Test kết nối ở máy in.
  4. Gỡ: xóa file %APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup\N2StorePrintBridge.vbs

Thêm máy in

Máy tem XP-470B dùng lệnh TSPL (tự gửi qua bridge). Gap = khe trống giữa 2 con tem trên cuộn — thường 2–3mm. Sai gap → in lệch / nhảy tem.
