Gửi ZNS (Zalo Notification Service)

Thông báo template chính thức tới bất kỳ SĐT nào (~200đ/tin). Cần kết nối OA ở tab Tài khoản trước. Template: 0 (đồng bộ ở thẻ OA).

Kết nối Zalo OA

Tạo app + OA tại developers.zalo.me, lấy App ID / Secret Key và Authorization Code (từ luồng OAuth của OA), rồi dán vào đây.
