Thêm tài khoản mới

💡 Token sẽ được tự động làm sạch (xóa khoảng trắng, prefix "jwt=", v.v.)
📋 Cách lấy JWT token:
  1. Mở tab mới: https://pancake.vn
  2. Đăng nhập vào tài khoản Pancake
  3. Mở DevTools (F12) → Console tab
  4. Gõ: document.cookie.split(';').find(c => c.includes('jwt'))
  5. Copy giá trị sau "jwt=" và paste vào ô trên

Page Access Tokens

Chưa có page token nào
Page token giúp gửi tin nhắn không bị giới hạn rate limit

Thêm Page Access Token

📋 Cách lấy Page Access Token:
  1. Vào Pancake → Chọn Page
  2. Vào Settings → Tools → API
  3. Copy "Page Access Token"
