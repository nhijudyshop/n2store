Privacy Policy

N2Store Messenger — Chrome Extension

Last updated: April 3, 2026

1. Overview

N2Store Messenger ("the Extension") is a browser extension designed for managing Facebook Page messages, integrated with the N2Store order management system. This policy explains how the Extension handles your data.

Key point: The Extension does NOT collect, store, or transmit your personal data to any third-party server. All data processing happens locally in your browser.

2. Data Collection

The Extension does not collect any of the following:

  • Personal identification information
  • Browsing history or activity outside of Facebook
  • Payment or financial information
  • Analytics or tracking data

The Extension accesses Facebook cookies solely to authenticate API requests to Facebook Messenger on your behalf. These cookies are never stored, copied, or transmitted outside of your browser.

3. Permissions Used

Permission Purpose
cookies Read Facebook authentication cookies to make authorized API requests to Facebook Messenger
storage Store extension settings and preferences locally on your device
notifications Display desktop notifications for new messages and order updates
alarms Schedule periodic checks for new messages
offscreen Process data in the background without visible windows
host_permissions
(facebook.com)		 Send and receive messages via Facebook's Messenger APIs directly from your browser

4. Data Storage

  • All settings are stored locally using Chrome's built-in storage API
  • No data is sent to external analytics, advertising, or tracking services
  • Message content is transmitted directly between your browser and Facebook's servers
  • The N2Store server (nhijudyshop.workers.dev) is used only for order management coordination — no message content passes through it

5. Third-Party Services

The Extension communicates only with:

  • Facebook / Meta — to send and receive messages via their official APIs
  • N2Store server (nhijudyshop.workers.dev) — for order data synchronization and real-time notifications (SSE)

No data is shared with any other third parties.

6. Security

All communication between the Extension and external services is encrypted using HTTPS. Facebook authentication credentials (cookies) are handled by Chrome's built-in cookie API and are never exposed to web pages or external scripts.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date. Continued use of the Extension after changes constitutes acceptance of the updated policy.

8. Contact

If you have questions about this Privacy Policy, please contact us via the GitHub Issues page.
