Ghi âm lưu trên portal OnCallCX

Bản ghi âm do FPT/OnCallCX lưu server-side (CDR — Call Detail Record). Truy cập trực tiếp qua portal: PBX → Calls (Lịch sử cuộc gọi) → Show detailed calls → chọn cuộc gọi → Play Audio / Download Audio.

OnCallCX chưa mở REST API public để lấy file audio. Bấm nút bên dưới để mở portal.
