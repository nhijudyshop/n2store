Lưu ý khi test thật trên Zalo:
• Đã commit/push lên GitHub Pages → đợi 2-4 phút deploy.
• Nếu Zalo đã cache URL cũ → thêm ?v=2 vào cuối URL khi gửi lại (vd: nhijudyshop.github.io/n2store/?v=2).
• Có thể debug card preview ở Facebook Sharing Debugger (Zalo dùng cùng chuẩn OG).
