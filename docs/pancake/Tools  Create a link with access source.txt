Tools > Create a link with access source
Shortlinks m.me or wa.me embedded with traffic source information help you identify which channel your customers come from. This allows you to measure effectiveness and optimize marketing campaigns for each corresponding channel.
Workflow
https://statics.pancake.vn/pancake-docs/2026/1/16/3914e995-f1bf-4c19-8b36-e08c9b5e496a_SCR-20260116-oofz.png
	•	Customer clicks on shortlink - Customer clicks on the  or  link created from Pancake
	•	Meta stores and sends data - Meta system stores encoded source information and sends it to Pancake system via webhook
	•	Pancake processes and displays - Pancake processes customer source information and displays it in the conversation management interface
How to Create Shortlinks
Step 1: Access the shortlink creation tool
	•	Log in with an account that has Admin permission
	•	In single page mode, go to Settings → Tools
	•	In the Create link with access source section, click the Add new button
https://statics.pancake.vn/pancake-docs/2026/1/16/9b304afd-dcc4-4b97-b233-570ec14bbdd9_SCR-20260116-oomq.png
Step 2: Configure information
Access Source
	•	You can customize the source name (e.g., Facebook Ads, Google Ads, Email Marketing)
	•	Pancake will display the conversation source according to this information in the management interface
	•	This information is only visible to admins, customers will not see it
Pre-filled Message (Required)
	•	Default message content that will be pre-filled when customers open the link
	•	Customers need to send this message for Pancake system to accurately identify the conversation source
https://statics.pancake.vn/pancake-docs/2026/1/16/e193395e-d402-4c77-a651-c280bb7d7eac_SCR-20260116-ophi.png
Step 3: Manage list
	•	The list of created sources will be displayed in the management table
	•	Only show the last 20 shortlinks per page
https://statics.pancake.vn/pancake-docs/2026/1/16/37315f5b-c512-4b3e-8c1b-b7c14abd1e07_SCR-20260116-opme.png
Display Conversation Source on Pancake
View conversation source information
Conversation source information will be displayed in the details section on the right side of the Pancake conversation interface. You can see the source that customers used to contact you (e.g., Facebook Ads, Google Ads, Email Marketing).
https://statics.pancake.vn/pancake-docs/2026/1/16/f62a9800-f01d-49c0-8f49-a51a37ac9aa5_SCR-20260116-oprd.png
Filter conversations by source
For effective management, you can use conversation filters by source:
	•	In the conversation list, click on the Filter icon
	•	Select Website Source
	•	Choose the specific source you want to view
