How to Create a Chat Plugin for Your Website
A step-by-step guide to setting up a live chat plugin for your website.
Step 1: Login to Pancake, choose Connect, select Chat Plugin
https://statics.pancake.vn/pancake-docs/2025/1/14/Screenshot 2025-01-14 at 16.53.55.png
Step 2: Create your Connection username
Step 3: Create your Display name - The name to display on Pancake dashboard
Connection username is in green 
Display name is in red
