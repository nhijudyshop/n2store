Pancake Conversation Tag Management
1. Add a new tag
https://statics.pancake.vn/pancake-docs/2025/7/31/Ảnh màn hình 2025-07-31 lúc 09.24.45.png
Thêm thẻ hội thoại 
Step 1: Click Add Tag
Step 2: In the pop-up window, fill the information:
	•	Index
	•	(The tag order number determines the display position of the tag in Conversations and in the tag management list. You can change the tag order by dragging and dropping.)
	•	Tag Name
	•	Select Tag Color
	•	Tag Description (optional)
Step 3: Click "Add Tag" to create a new tag
2. Edit, Delete a tag
When hovering over a tag in the list, you will see icons such as Edit tag and Delete tag
 You can delete multiple tags at once by selecting them and then clicking Delete.

3. Rearranging Tag Positions
To rearrange tag positions, hover your mouse over the first column and drag and drop.
Note: This action will also update the tag’s index

4. Sync/Copy conversation tag
To sync/copy tag from page A to page B. Follow these steps:
https://statics.pancake.vn/pancake-docs/2025/7/31/Ảnh màn hình 2025-07-31 lúc 09.29.42.png
Step 1: Click the Copy icon next to the Add Tag button
Step 2: Select the destination page (Page B)
The destination page (Page B) is the page where you want to copy the tags from the source page (Page A).
Step 3: Select the tags (from Page A) that you want to sync to Page B
Step 4: Choose to either Add to existing list or Replace existing list
	•	Add to existing list: If you select this option, the tags you choose to sync from Page A will be added to the end of the tag list on Page B. All existing tags on Page B will be retained.
	•	Replace existing list: If you select this option, the tags you choose to sync from Page A will completely replace the current tags on Page B. All existing tags on Page B will be DELETED. Please use this option with caution!
Step 5: Click Copy to begin the tag copying process
5. Deactivate a Tag
If you want to stop using a tag without deleting it, you can use this option. Tags marked as deactivated will no longer appear in the tag list within Conversations.
However, conversations that were previously tagged with it will remain tagged, and you can still filter past conversations using this tag as usual.
