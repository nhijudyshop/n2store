Conversation Tag Settings
1. Functions and Tools 
1.1 Attach tags to a conversation 
Conversations can be assigned multiple tags when this option is enabled. If you turn off this feature, each conversation will be limited to only one tag. Previously tagged conversations will remain unchanged.
1.2 Show full tag names
Show full tag names in the conversation list: When this option is enabled, the conversation tags displayed in the conversation list will show their full names instead of just colored dots.

1.3 Multi-tag Filter Mode
Filter multiple tags simultaneously using OR (A or B) and AND (A and B) conditions: When this option is enabled, you can filter conversations by combining multiple tags using logical conditions.

1.4 Synchronized Tagging
All conversations from the same customer (including comments, messages, etc.) will be automatically assigned the same tags when this option is enabled. Tags that were assigned before enabling this setting will remain unchanged.

1.5 Call Tags
	•	Tag conversation when call is unsuccessful: You can select a tag for Pancake to automatically assign to the conversation when a call to the customer fails.
	•	Tag conversation when call is not answered: You can select a tag for Pancake to automatically assign to the customer’s conversation when the customer does not answer the call.
