Call
The Call feature lets you make and receive calls directly on Pancake, including from both Messenger and WhatsApp. This way, you don’t need to switch between apps and can keep your entire contact history and customer support in one place.
https://statics.pancake.vn/pancake-docs/2025/9/25/ae60d66d-cadd-430f-b9dd-e3855cc23306_iScreen Shoter - Google Chrome - 250925121704.jpg
A. Set up and use calls on Messenger
To enable the Call feature in conversations, go to Settings → Calls. Here, depending on your Page, the Messenger Call feature may or may not be available.
	•	Case 1: If your Page is not supported, the screen will display a message: “Your Page does not support the call feature” (see example below).

	•	Case 2: If your Page is supported, you will see the option to set up the call feature (as shown in the example below).
https://statics.pancake.vn/pancake-docs/2025/9/25/8138bc7c-2e8a-4fa9-9a82-4a81f1e7fead_iScreen Shoter - Google Chrome - 250925122044.jpg
Here, you can choose to make calls using either Pancake or Meta Business Suite. The details of how each option works will be explained in the sections below.
1. Make calls from Pancake
When you set up calls in Pancake, you can either make a direct call or send a call request. You’ll also have access to advanced options such as setting available call hours, assigning staff, or adding automatic tags.
1.1 Call settings

Here, you can set up outgoing calls in 2 ways:
	•	Sent call request: When you press the Call button, the system will always send the call request to the customer. The customer can then either accept or decline your request.

- You can send a maximum of 2 call requests per day to the same customer. 
- If the customer calls you first or has already accepted a call request, you can make direct calls to them for the next 7 days without sending another request.

	•	Initiate a call: When you press the Call button, the system will start a direct call to the customer. If the customer has not previously given permission, the system will automatically send a call request before making the connection.

1.2 Advanced features
In this section, you can use some advanced call features:
	•	Call routing: When a call comes in, the system can prioritize ringing for: Priority for assigned account/ Ring all accounts/ Selected account.
	•	Auto add tags to conversation: You can set up automatic tagging for conversations when: After receiving incoming call success and After having a missed call. This helps you easily track and manage customers for better follow-up and support.

2. Making calls from Meta Business Suite
When setting up calls through Meta Business Suite, you can only make calls using the Sent call request option. Advanced features are not available in this mode.

B. Set up and use calls on WhatsApp
1. Calls setting
To set up the “Call” feature in a conversation, go to Settings → Calls, then select WhatsApp to start using it.
Note:
Some countries may not support WhatsApp calls. Check the details for more information here
	•	Case 1: If your Page is not supported, the screen will display: “Your Page does not support the call feature” (see example below).
