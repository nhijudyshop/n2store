General
Instructions and explanation of the meaning/functions of general settings for the page.
Some settings may vary depending on the platform you are using.

1. Notifications and Conversations
1.1 Sound and Notification
	•	Browser notifications for new messages or comments: Whenever there is a new message or comment from Facebook, you will receive a notification via your browser.
	•	Play sound for new conversations: A sound will be played whenever there is a new notification.
1.2 Conversations
	•	Move unread conversations to the top of the list: Prioritize displaying unread conversations at the top of the conversation list.
	•	Quickly jump to the next unread message in the list: You can quickly switch to the next unread message in the list by pressing the Enter key twice in a row.
2. Automated functions

2.1 Auto-hide comments
	•	Hide comments on posts (except for comments configured in the Posts section):
	•	Pancake can automatically hide comments on posts. You can choose which types of comments to hide, such as those containing phone numbers, those without phone numbers, or all comments. However, if you have set specific comment hiding rules for a particular post in the Posts section, those settings will override this general setting for that post.
	•	Previously hidden comments will be shown again on all posts: Pancake can automatically unhide comments after a selected period of time (X minutes/hours), based on your configuration.
	•	Hide all spam comments: Pancake will automatically hide comments that are identified by the system as spam when this setting is enabled.
2.2 Automatically ignore comments that tag friends
	•	Ignore messages if the new message is a comment tagging friends:
	•	Pancake can ignore comments where customers tag their friends. You can enable this option and choose to ignore comments that contain only tagged names or those that include both tagged names and additional content.
	•	Automatically like comments where customers tag friends:
	•	You can enable this option to let Pancake automatically like comments in which customers tag their friends, based on the settings above.
2.3 Automatically ignore new messages that are stickers
If the customer's new message is a sticker, the system will automatically ignore it. Pancake will skip processing sticker messages when this option is enabled.
2.4 Auto-save Birthday
	•	Detect and save the birthday of customers who send messages: When a customer sends a message, Pancake will automatically detect and save their birthday if this option is enabled.
2.5 Auto-like comments 
Automatically like customer comments when replying: If this option is enabled, each time you reply to a customer’s comment, Pancake will automatically like that comment on your behalf.
2.6 Send events to Facebook
Send the Lead Submitted event to Facebook to optimize ads: You can enable this option to optimize the Lead Submitted metric in Facebook Ads Manager. Whenever a customer leaves their phone number, Pancake will send an event to Facebook to help optimize your ads toward audiences with similar behavior.
2.7 Ignore Spam Folder
By default, the system scans all messages, including those in the Spam folder. If you prefer the system not to scan Spam messages, enable this option.
The Spam folder on Facebook contains messages that Facebook suspects to be spam. However, some legitimate customer messages may also be mistakenly placed in this folder.

3.2 Real-time Botcake Tag Synchronization
When adding or removing customer tags in Pancake, automatically sync them with Botcake:
If this option is enabled, customer conversation tags between Pancake and Botcake will be synchronized in real-time whenever changes are made.
3.3 Automatically Create Orders in POS
New orders will be automatically created in the POS system when a customer's message contains a phone number:
If this option is enabled, whenever a customer's message includes a phone number, Pancake will automatically create an order in Pancake POS.
