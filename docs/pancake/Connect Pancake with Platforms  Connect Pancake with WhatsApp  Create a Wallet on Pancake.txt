Connect Pancake with Platforms > Connect Pancake with WhatsApp > Create a Wallet on Pancake
To use the WhatsApp API, once you've connected Pancake with the WhatsApp API, creating a prepaid Pancake Wallet is required. Follow the steps below to create your Pancake Wallet.
1. Create a Pancake Wallet
1.1 Definition
What is a Pancake Wallet?
The Pancake Wallet is a digital wallet in the Pancake system, allowing you to preload funds to automatically pay for Template fees when using the WhatsApp API.
This ensures that messaging with customers is seamless and uninterrupted due to insufficient balance.
	•	Why do you need to create a Pancake Wallet?
To start a conversation through the WhatsApp API, you need to use Templates, and each template incurs a fee. The Pancake Wallet is required and lets you add funds in advance so the system can automatically pay when sending templates, ensuring messages are delivered smoothly without interruption.
1.2 Create a Pancake Wallet
To use the WhatsApp API, creating a prepaid Pancake Wallet is a required step you must complete first.
✅ Step 1: If you haven’t connected your account to Pancake yet, you need to do so first.
In Pancake, go to “Account Management” → “Connect”

After that, you will be redirected to Pancake's Account page.
Click on “Sign Up” or “Continue with Pancake” (in case you were already logged in) to create your wallet account. This account will be registered using your email address.

✅ Step 2: Go to "Pancake Wallet ID" → Click on "+ Create New" to add a new wallet.
Then, enter your wallet name and make sure to always select USD as the currency.

✅ Step 3: In the “Add Card” step, you are not required to enter any card information.
You can skip this step and move on to the next one.
1. Pancake will not save your card information. 
2. It’s recommended to use a card for a smoother experience and to avoid any service interruptions.

Case 1: Not using a card → Select “Continue” to move to the “Top-up” step.
Since you're not using a card, the system will automatically set the payment method to “Bank Transfer” by default.

Case 2: Using a Card → Enter your card information, and the payment will be processed via "Stripe".
Click “Link Card” to confirm the card connection.
