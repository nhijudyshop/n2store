Configuration for your Chat Plugin
The following guide will help you install the Chat Plugin on your website to receive conversations.
You can see a Preview on the right
1. Display Settings
Go to Setting > Chat Plugin 
https://statics.pancake.vn/pancake-docs/2025/1/14/Screenshot 2025-01-14 at 17.30.09.png
1. Button Display Style
In this setting, three button styles are currently supported for display on your website: round, vertical, and horizontal. Choose the style you prefer.
Button Text: For vertical or horizontal button styles, you can add custom text to the button.
2. Button Size and Position
Depending on the button style, you can customize its size and position. Specifically:
	•	Round Button: Chat button size, alignment type, horizontal margin, bottom margin.
	•	Horizontal Button: Alignment, margin, height.
	•	Vertical Button: Alignment, top margin on desktop, bottom margin on mobile, button width.
https://statics.pancake.vn/pancake-docs/2025/1/14/Screenshot 2025-01-14 at 17.32.08.png
3. Colors
Primary Color: The main color theme for your Chat Plugin. You can choose a preset color or enter a custom hex code.
Text Color: Select a text color that matches and complements the primary color.
4. Button Icon
Choose an icon for the Chat Plugin button. You can either use a preset icon or upload a custom image.
https://statics.pancake.vn/pancake-docs/2025/1/14/Screenshot 2025-01-14 at 17.34.05.png
  5. Page Avatar
Page Name: The name that will be displayed to customers when they view your chat.
Page Avatar: Set the avatar image that represents your page during conversations with customers.
6. Chat Window Height
You can customize the height of the chat window separately for mobile and desktop devices.
2. Form
https://statics.pancake.vn/pancake-docs/2025/1/14/Screenshot 2025-01-14 at 17.36.41.png
Form Title: Your welcome message.
Create New Field: In addition to the default fields, you can add custom fields based on your needs.
Explanation of the columns in the table:
	•	Key: The name of the information field.
	•	Type: The data type of the field. You can choose from: Number, Text, Long Text, or Dropdown.
	•	Label: The display name of the field shown on the Chat Plugin.
	•	Placeholder: A hint or example of what needs to be entered. You can describe it to help customers understand what to fill in.
	•	Display: Toggle to show or hide the field.
	•	Required: If checked, customers must fill in this field before starting the chat. For example, you can require a phone number.
	•	Actions: For custom fields, you can delete them using the delete button in this column.
3. Functions
Switch to the Functions tab to configure additional detailed settings.
https://statics.pancake.vn/pancake-docs/2025/1/14/Screenshot 2025-01-14 at 17.37.15.png
1. Voice Call Feature
This feature allows customers to initiate voice calls from the Chat Plugin to your page completely free of charge. During the Beta period, the system will automatically apply a trial package with the following features:
	•	1 concurrent call.
	•	Up to 20 calls per day.
	•	Calls are prioritized to staff assigned to the conversation.
	•	Each call can last up to 15 minutes.
	•	If a call is missed or declined, the customer must wait 5 minutes before making another call.
https://statics.pancake.vn/pancake-docs/2025/4/25/Untitled design (1).png
2. Button Animation
Enable button animation to attract more attention from customers.
3. Auto Open
You can set the Chat Plugin to open automatically after a specified number of seconds when a customer visits your website. This setting can be turned off, enabled only on mobile, only on desktop, or on all devices.
4. New Message Notification
Enable this option to allow customers to receive notifications when new messages arrive.
5. Language
Select the default language to be displayed on the Chat Plugin. You can customize language-specific content in the Form section.
6. Quick Replies
Allows you to configure quick reply flows using Botcake.
7. Support Service Rating
Configure the customer support rating feature to evaluate staff performance and track customer satisfaction for each support agent.
4. Plugin Code
Instructions for installing the Chat Plugin on your website.
https://statics.pancake.vn/pancake-docs/2025/1/14/Screenshot 2025-01-14 at 17.42.04.png
1. Domain Setup
Whitelist the list of domains where you intend to embed the Chat Plugin. This helps prevent the <script> tag from being used on unauthorized websites.
You must declare all domains you plan to use. For example: pancake.vn, botcake.io, etc.
2. Plugin Installation Code
To display the Chat Plugin on your website, you need to insert the provided <script> code into the <body> tag of your website’s HTML.
In addition, Pancake allows you to add custom configurations to optimize the Chat Plugin for your specific needs, such as:
	•	Changing the language for each website.
	•	Opening the chat window automatically.
	•	Pre-filling form data if customer information is already available.Form Title: Your welcome message.
	•	Create New Field: In addition to the existing fields, you can add custom fields as needed.
Explanation of the columns in the table:
	•	Key: The field name.
	•	Type: The data type of the field. You can choose from: Number, Text, Long Text, or Options.
	•	Label: The display name of the field in the Chat Plugin.
	•	Placeholder: A hint or example of the information to be entered. You can describe what the customer needs to input.
	•	Visible: Turn the field on or off.
	•	Required: If checked, the customer must enter this information before starting the chat. For example, you can make it mandatory for customers to enter their phone number.
	•	Action: For custom fields, you can delete them using the delete button in this column.
