Guide to connecting Pancake with LineOA
Requirements: 
- You must have a Line OA account and be the Admin of that account. 
- You must have a Pancake account. 
- You must have installed the Pancake v2 extension (refer to the guide on how to install the Pancake v2 extension).

At this point, you have successfully connected Pancake to LineOA.
Some Notes on Using Line OA with Pancake: 
- The number of messages in Pancake is calculated based on the number of broadcast messages sent via Line. Therefore, after the 1,000 free broadcast messages per month, customers must purchase a broadcast plan on Line to continue sending messages through Pancake. 
- After connecting to Line OA, messages will only be received in Pancake starting from the time of connection. Currently, Line does not support syncing past messages. 
- Once connected, the admin should only use Pancake to respond to customers. If you actively send messages to customers via the Line OA app (not through Pancake), those messages will not appear in Pancake.
