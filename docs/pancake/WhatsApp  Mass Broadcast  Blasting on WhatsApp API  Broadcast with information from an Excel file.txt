WhatsApp > Mass Broadcast / Blasting on WhatsApp API > Broadcast with information from an Excel file
This is an elevated version of the previous guide - using variation.
Using WhatsApp API, you can broadcast mass messages to the client's WhatsApp number.
Note: While broadcasting via WhatsApp may seem convenient, it's crucial to adhere to best practices. Your WhatsApp API number may be blocked if clients report or block your broadcasts. Avoid spamming clients, refrain from sending inappropriate content, and always provide an option for clients to unsubscribe from your broadcast stream.
1. Preparation
To do a proper broadcast, please ensure that your WhatsApp number is working properly on Botcake. If you do not see your number appear here, please click the "Add new page" button and connect it.
https://statics.pancake.vn/pancake-docs/2025/3/24/imagee8dee4c4-9636-496b-9948-943df63009f1.png
Next, prepare an Excel file with the information you would like to broadcast. In this Excel file, the first column must always be the client's phone number with the country code.
https://statics.pancake.vn/pancake-docs/2025/3/25/imagee127732f-21bf-4f34-b1e7-16ff8b5e697f.png
Finally, head to WhatsApp Manager to create a template with variables.
 If you do not know how to create a template, 
Please make sure these variations match the columns you have in the Excel file prepared.
https://statics.pancake.vn/pancake-docs/2025/3/25/image43ab7395-6e1d-46a1-b597-a9cb0512cd29.png
For example: in the Excel file above I have 5 different pieces of information to be included in the message: last name, Full name, Location, Registration Date, and Timeslot. So in the template, I must create 5 corresponding variables. When broadcast in Botcake, you will need to map these variations by their column name (in the next step).
2. Broadcast away!
To create a broadcast campaign, choose “Broadcasts” on the toolbar → “+ New message”
https://statics.pancake.vn/pancake-docs/2025/3/25/image7beb2af7-add8-45c6-801f-618b8842f28b.png
On the “New messages”, delete the default content and choose “Templates”.
https://statics.pancake.vn/pancake-docs/2025/3/25/image31d5f3e9-2dab-4a77-86df-3343e0321389.png
Choose the approved template that you prepared from the list. Then, fill in the column names in the corresponding parameters (variables). Please remember to copy the EXACT column name, and keep them between two braces {{ }}
https://statics.pancake.vn/pancake-docs/2025/3/25/image6744b108-97ac-47fa-8d88-677cf2a47d73.png
Sending templates means opening a new 24h conversation, you’ll be charged once the templates are sent successfully, and the fee depends on the type of your template. More pricing details are here.
After the setting is complete, it should look like this. Please check the template's content carefully, then click “Settings” or “Next step”
https://statics.pancake.vn/pancake-docs/2025/3/25/imaged3ec493c-2422-4716-ad21-d1a6181c4035.png
On the Broadcast setting tab, choose the third option and upload the excel file you prepared. Please double-check the file to make sure the phone numbers include the country code, and the information you want to broadcast is correct.
https://statics.pancake.vn/pancake-docs/2025/3/25/image2d6a604a-1fa2-4e58-a812-5baafd98252d.png
For the “Number of customers” and “Maximum frequency of sending” sections you can use the default settings.
https://statics.pancake.vn/pancake-docs/2025/3/25/image3e0e04a1-b727-45d8-8585-38a71749618a.png
Finally, choose “Send now” to send mass messages to your customers.
https://statics.pancake.vn/pancake-docs/2025/3/25/imagef1da5982-513e-4deb-be49-356f6dcfa803.png
The template will be sent directly to the recipients in the Excel file.
https://statics.pancake.vn/pancake-docs/2025/3/25/image30542e52-aafb-4ac8-bc39-31248a4c2e43.png
Congratulation! You have just completed a broadcast campaign on Botcake!
