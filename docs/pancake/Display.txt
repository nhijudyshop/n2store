1. Display Main Assigned Account
Show the main assigned account in the conversation list: When this option is enabled, the account assigned as the main handler for the conversation will be displayed in the conversation list, as shown in the highlighted area below.

2. Display account currently viewing the message
Show the account currently viewing the message in the conversation list: When this option is enabled, you will see which account is currently viewing the message in the conversation list, as shown in the highlighted area below.

3. Page color
Page color will help you quickly distinguish messages from different pages when using the merged pages mode. The selected color will appear in two places:
	•	In the page list
	•	In the conversation list, before each conversation
