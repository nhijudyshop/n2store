WhatsApp > Mass Broadcast / Blasting on WhatsApp API
Broadcast WITHOUT variables in the template.
After connecting the WhatsApp API in Pancake, the WhatsApp number will be automatically synced to Botcake. If the number doesn’t appear, click on “Add New Page” → Select the WhatsApp platform → Activate the page.
