User
This section provides an overview of employee performance, helping you track activities such as customer interactions, response times, and work results. The data can be exported as an Excel file for easier tracking and analysis.
1. Date range
You can select your desired period (by day, week, month, year, etc.) to view the corresponding data statistics.
https://statics.pancake.vn/pancake-docs/2025/4/27/iScreen Shoter - Google Chrome - 250427171012.jpg
🧠 Example: The statistics in the User will be filtered for this month, specifically from April 1, 2025, to April 27, 2025.
The time frame selected in the "Overview" section may differ from that in the "User" section. Please double-check to ensure you're viewing the correct data.
2. Analytics charts
The statistical charts show the Top 5 productive buddies and the Response ratio by employee. In there:
Top 5 productive buddies: This list includes the 5 employees with the shortest average response times, calculated from when a customer sends a message or comment to the Fanpage until the employee responds.
Note: 
Only counts within working hours as set in the "Working Hours" section in General Settings
	•	Response ratio: This shows the response rate of each employee compared to the total responses from all employees.
https://statics.pancake.vn/pancake-docs/2025/4/27/iScreen Shoter - Google Chrome - 250427171102.jpg
🧠 Example: The chart shows the interactions of all staff in this month (April 1, 2025 – April 21, 2025)
🔍 To understand the definition of each metric, please refer to the Detailed user statistics section below.
