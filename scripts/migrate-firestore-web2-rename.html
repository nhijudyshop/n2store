Web 2.0 Firestore Migration

Quy trình 3 bước: Backup OLD → Delete OLD → Seed fake data NEW. Vì Web 2.0 đang ở chế độ test, có thể xoá dữ liệu OLD và bơm dữ liệu giả vào NEW để demo / test.

⚠️ Script này XOÁ dữ liệu OLD vĩnh viễn sau khi backup. Chạy ĐÚNG THỨ TỰ: 1 → 2 → 3.

1 Backup OLD → JSON file

Đọc cả 4 OLD collection, gói thành 1 file JSON, tự download xuống máy. File này là snapshot để khôi phục nếu cần.

2 Delete OLD collections

Xoá doc main trong cả 4 OLD collection. Chỉ bấm sau khi Backup xong.

3 Seed fake data vào NEW

Bơm dữ liệu giả vào web2_so_order, web2_supplier_wallet, web2_customer_wallet, web2_suppliers để test Web 2.0. Dataset gồm: 3 tab NCC (VN, China, Korea), 5 NCC, 6 shipments với 18 line items, ví NCC với payment history, ví KH với 4 khách + transactions, danh sách NCC với note.
